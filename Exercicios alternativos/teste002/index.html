HTML: uma linguagem de programação Marcação

Historia

O Hypertext Markup Lenguage, mais conhecido como HTML, foi criado por Tim Berners Lee na decada de 90, no CERN (European Council for Nuclear Research) na suíça. Inicialmente Tim, projetou o HTML para interligar intituições de pesquisas proximas, para que elas conseguissem compartilhar documentos com mais facilidade, após sua criação, quando a biblioteca de desenvolvimento WWW (World Wide Web) teve seu alcance expandido mundialmente pelo proprio Tim, o HTML se tornou de uso global na WEB

Como funciona

O HTML, como ja o proprio nome diz, é uma linguagem de marcação, ela é ultilizada com o principal foco no conteudo, ou seja, todo tipo de conteudo dos sites são adicionadas em HTML...
inclusive esse texto
Mas então, como este conteudo é adicionado?
Simples! (na verdade não), a linguagem HTML ultiliza TAGs, essas TAGs são responsaveis por adicionar titulos, paragrafos, caracteres especiais, imagens, marcações no texto, emojis 😀 e varias outras coisas. Cada Tag é responsavel por uma função especifica, por exemplo, a TAG <p> adiciona um paragrafo, ja a tag <mark> faz o texto ficar marcado, do mesmo jeito que um marca texto que é ultilizado nas escolas, como você pode ver, as TAGs são letras ou palavras entre < e >.
Então se a linguagem HTML adiciona o conteudo, ela tambem torna o site bonito?
Não, essa linguagem fica responsavel, como ja falamos, pelo conteudo do site, por exemplo: no site do g1.com todas as noticias são adicionadas com HTML, assim como os titulos dessas noticias, pois é um conteudo, agora, respondendo a pergunta, quem cuida do design do site é o CSS...
